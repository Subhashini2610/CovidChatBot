# CovidChatBot
 
This is a sample implementation of a ChatBot that provides all stats about Covid19.

The Covid data is refreshed once every 24 hours.

## COVID Data Sources: 

https://github.com/microsoft/Bing-COVID-19-Data

www.bing.com/covid
